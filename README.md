# mips-face-detect

SpringBoot RESTful API 脚手架

## 简介

这是一个基于`SpringBoot 2.1.1 RELEASE`，用于搭建`RESTful API`工程的脚手架，只需`三分钟`你就可以开始编写业务代码，不再烦恼于构建项目与风格统一。


## 基本功能

- 封装了 RESTful 风格的返回结果
- 全局接管异常，并使用 RESTful 风格返回异常信息
- 简单封装了 jjwt，提供了一个开箱即用的认证模块
- @Auth 注解，用注解的方式校验用户信息
- 集成了常用的库，详见相关依赖
- 集成了 MybatisPlus

## 相关依赖

- SpringBoot
- MybatisPlus
- lombok
- jjwt
- fastjson
- druid
